Post 5 - On Classes

Classes are an exciting feature of Ruby. A class can be useful when you need to do something over and over.

My best example is classifying animals.

What if we needed to determine what breed of dog we have based on a number of characteristics? For example, if we have a tall, long-haired, working breed, it could be a German Shepherd (could be) but if we have a short, long-haired, toy breed dog, it could be a toy poodle (again, could be - we would have to add a large number of instance variables like height, weight, breed age, friendliness, and so on and so forth to really get into it)

Happy Coding
